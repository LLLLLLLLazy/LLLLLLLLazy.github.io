# Jiaoyang Yan's Academic Homepage

Source for [LLLLLLLLazy.github.io](https://LLLLLLLLazy.github.io/).

The site highlights education, research experience, selected systems projects, and honors. Public contact details are intentionally limited to a university email address.

## Local development

```bash
bundle install --path vendor/bundle
bundle exec jekyll serve
```

The generated site is available at `http://localhost:4000`.

## Theme

Built with the open-source [Minimal Light](https://github.com/yaoyao-liu/minimal-light) Jekyll theme, licensed under CC0 1.0.
